# Weather App

## Description

A Python application that fetches and displays real-time weather information for a user-specified city using a weather API.

## Features

* Accepts city name as input
* Fetches live weather data from an API
* Displays:

  * Temperature
  * Humidity
  * Weather Condition
* Easy to use

## Technologies Used

* Python
* requests module
* wttr.in Weather API

## How to Run

1. Install the required library:

   ```
   pip install requests
   ```
2. Run `weather_app.py`.
3. Enter the city name.
4. View the current weather details.
